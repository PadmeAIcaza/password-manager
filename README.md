# Functionality
-  It saves the email and passowrd used for a website, as well as the website's name
-  Every time the "Add" button gets clicked (if all field are non-empty) it gives a pop-up confirming the details.
-  If all details are correct, user clicks the "Ok" button on the pop-up, and the details get saved on the file. Otherwise, the user can click the "Cancel" button, and the details will not get saved on the file.
-  Once the details got saved on the file, the fields get emptied automatically
-  If at least one of the fields are empty, it will give a pop-up error asking to not leave empty fields
-  There is a "Generate Password" functionality, where it will generate a password once the button is clicked
